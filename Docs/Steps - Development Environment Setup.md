# Steps - Development Environment Setup.md
- This document details how this Sunvoy Web Application Development Environment was setup.


## 1. This application was created using npx

```
    npx create-next-app@latest
```

Note: Enable everything.

## 2. Install pnpm. It prefers to use pnpm.

```
    npm i -g pnpm
```

## 3. Install modules    

```
    pnpm install

    or 

    pnpm i

```
### Note: See package.json for packages installed or used in this application. You can also run the command: pnpm list
